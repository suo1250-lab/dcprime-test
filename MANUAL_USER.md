## 1. 시스템 소개

DCPRIME은 시험지 이미지를 AI가 자동으로 채점해 주는 학원 전용 자동채점 시스템입니다.

### 주요 기능 한눈에 보기

| 기능 | 방법 
|------|------
| 영어 단어시험 자동채점 | NAS 미채점 폴더에 파일 복사 |
| 입학테스트 자동채점 | NAS 미채점 폴더에 파일 복사 |
| 수학 OMR 자동채점 | NAS 미채점 폴더 또는 웹 직접 업로드 |
| 답지 자동 등록 | NAS 답지 폴더에 PDF 복사 |
| 학생/반 관리 | 웹 관리 페이지 |
| 성적 통계 및 엑셀 내보내기 | 웹 관리 페이지 |

### 웹 주소 안내

관리자에게 받은 웹 주소로 접속합니다. 

| 페이지 | 경로 |
|--------|------|
| 대시보드 (메인) | `/` |
| 학생 관리 | `/students` |
| 단어시험 채점 검토 | `/word-submissions` |
| 단어시험 목록/관리 | `/word-tests` |
| 입학테스트 결과 | `/results` |
| 수학 OMR 결과 | `/math-history` |

## 2. 단어시험 채점하는 법

### 전체 흐름

```
① 시험지 스캔/촬영
      ↓
② NAS 미채점 폴더에 파일 복사
      ↓
③ AI 자동 채점 (자동, 1~2분 소요)
      ↓
④ 웹에서 채점 결과 검토
      ↓
⑤ 수정 후 확정
      ↓
⑥ 빨간펜 PDF 자동 생성 → NAS 채점완료 폴더 저장
```

---

### 2-1. 파일 준비 및 NAS 폴더에 넣기

**지원 파일 형식:** JPG, PNG, WEBP, PDF

**파일 이름 규칙 (중요):**
- AI가 파일 이름에서 학생 이름을 자동으로 인식합니다.
- 파일 이름에 반드시 **학생 이름**을 포함해 주세요.
- 예시: `홍길동.jpg`, `20260410_홍길동.pdf`, `중2_홍길동_단어시험.jpg`

**NAS 폴더 경로:**
```
자동채점기/영어단어튜터링/미채점/
```

- NAS(시놀로지)에 접속한 뒤 위 경로에 파일을 복사/붙여넣기 합니다.
- 한 번에 여러 파일을 넣어도 됩니다. (수십개 넣으면 서버 다운됩니다 ㅠㅠ) 

> **주의:** 파일이 완전히 복사된 후 시스템이 감지합니다. 복사 중간에 처리될 수 있으므로, 대용량 PDF는 복사가 끝난 것을 확인한 뒤 다음 단계로 넘어가세요.

---

### 2-2. AI 자동 채점 대기

파일을 폴더에 넣으면 백엔드 시스템이 자동으로 감지하여 채점을 시작합니다.

- 처리 시간: 이미지 1장 기준 약 30초~1분
- 채점 중에는 별도로 할 일이 없습니다.
- 채점이 완료되면 웹 페이지(`/word-submissions`)에 결과가 나타납니다.

---

### 2-3. 웹에서 채점 결과 검토

1. 웹 브라우저에서 `/word-submissions` 페이지로 이동합니다.
2. 목록에서 검토할 학생의 채점 결과를 클릭합니다.
3. AI가 판독한 학생 답안과 정오 여부를 확인합니다.
4. 잘못 판독된 항목이 있으면 직접 수정합니다:
   - 학생 답안 텍스트를 수정할 수 있습니다.
   - O/X 버튼으로 정오 여부를 바꿀 수 있습니다.
5. 학생 이름이 잘못 인식된 경우 이름을 직접 수정합니다.

**상태 표시 안내:**

| 상태 | 의미 |
|------|------|
| `pending_manual` | AI 채점 전 (수동 처리 대기) |
| `pending_review` | AI 채점 완료, 선생님 검토 대기 |
| `confirmed` | 선생님이 확정 완료 |

---

### 2-4. 채점 확정 및 빨간펜 PDF

1. 검토가 끝나면 **[확정]** 버튼을 누릅니다.
2. 확정 즉시 빨간펜 채점 결과 PDF가 자동으로 생성됩니다.
3. PDF는 NAS의 `채점완료/` 폴더에 자동 저장됩니다.
   - 파일명 예시: `20260410_A반_홍길동.pdf`
4. 웹에서도 **[PDF 다운로드]** 버튼으로 직접 다운받을 수 있습니다.

**빨간펜 PDF에 포함되는 내용:**
- 학생 이름, 시험명, 학년
- 최종 점수 및 정답률
- 틀린 문항 번호 목록
- 문항별 O/X 표시 및 정답 안내

> 확정 후 수정이 필요한 경우: **[다시 열기]** 버튼을 누르면 `pending_review` 상태로 돌아갑니다.

## 3. 입학테스트 채점하는 법

단어시험과 동일한 흐름으로 처리됩니다.

**NAS 폴더 경로:**
```
자동채점기/입학테스트/미채점/
```

**채점 완료 저장 경로:**
```
자동채점기/입학테스트/채점완료/
```

웹에서는 `/results` 페이지에서 결과를 확인합니다.

## 4. 답지 등록하는 법

답지 PDF를 NAS 지정 폴더에 넣으면 시스템이 자동으로 파싱하여 등록합니다. 웹에서 직접 입력할 필요가 없습니다.

### 4-1. 영어 단어시험 답지

**폴더 경로:**
```
자동채점기/영어단어튜터링/영어단어테스트답지모음/
```

- 답지 PDF를 위 폴더에 복사합니다.
- 시스템이 자동으로 문항 번호, 문제(영단어), 정답(뜻)을 파싱합니다.
- 등록된 답지는 웹 `/word-tests` 페이지에서 확인할 수 있습니다.

### 4-2. 입학테스트 답지

**폴더 경로:**
```
자동채점기/입학테스트/입학테스트답지모음/
```

### 4-3. 수학 OMR 답지

**폴더 경로:**
```
자동채점기/수학/수학답지모음(주테omr전용)/
```

- 수학 OMR용 답지(정답 번호 배열)를 위 폴더에 넣습니다.

### 답지 등록 확인 방법

웹 `/word-tests` 페이지에서 시험 목록이 생성되었는지 확인합니다. 등록에 실패한 경우 파일이 NAS `미채점오류/` 폴더로 이동됩니다.

## 5. 수학 OMR 채점하는 법

수학은 OMR(마킹 시트) 방식의 객관식 채점을 지원합니다.

### 방법 1: NAS 폴더 자동처리

**폴더 경로:**
```
자동채점기/수학/미채점/
```

- 스캔한 OMR 시험지 파일을 위 폴더에 넣습니다.
- 파일명에 학생 이름을 포함해야 합니다.
- 채점 완료 후 `/math-history` 페이지에서 결과를 확인합니다.

### 방법 2: 웹 직접 업로드

1. `/math-submissions` 페이지로 이동합니다.
2. **[파일 업로드]** 버튼을 클릭합니다.
3. 시험지 이미지 파일을 선택하고 학생 정보를 입력합니다.
4. 업로드 후 AI가 자동으로 마킹된 번호를 인식하여 채점합니다.

### 채점 완료 저장 경로

```
자동채점기/수학/채점완료/
```

## 6. 학생 및 반 관리

### 6-1. 학생 등록 (개별)

1. 웹 `/students` 페이지로 이동합니다.
2. **[학생 추가]** 버튼을 클릭합니다.
3. 이름, 학년, 학교, 반, 전화번호, 담당 선생님을 입력합니다.
4. **[저장]** 버튼을 누릅니다.

**입력 항목 안내:**

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 이름 | 필수 | 시험지 파일명과 일치해야 AI가 자동 매칭 |
| 학년 | 필수 | 예: 중1, 중2, 고1 |
| 학교 | 선택 | |
| 반 | 선택 | 채점 결과 PDF 파일명에 반영됨 |
| 전화번호 | 선택 | |
| 담당 선생님 | 선택 | |

### 6-2. 학생 일괄 등록 (엑셀)

여러 학생을 한 번에 등록할 때 사용합니다.

1. `/students` 페이지에서 **[엑셀 템플릿 다운로드]** 버튼을 클릭합니다.
2. 다운받은 템플릿에 학생 정보를 입력합니다.

   **엑셀 컬럼 순서:**
   ```
   이름 | 학년 | 학교 | 반이름 | 전화번호 | 담당선생님
   ```

3. 저장 후 **[엑셀로 일괄 등록]** 버튼을 눌러 파일을 업로드합니다.
4. 동일한 이름+학년 학생이 이미 있으면 정보가 업데이트됩니다.

### 6-3. 반(Class) 관리

- 반은 학생 등록 시 `반이름` 컬럼에 입력하면 자동으로 생성됩니다.
- 웹 `/classes` 페이지에서 반 목록 확인 및 관리가 가능합니다.

### 6-4. 학생 상세 프로필

`/students` 목록에서 학생 이름을 클릭하면 해당 학생의 상세 프로필 페이지로 이동합니다.

**프로필 페이지에서 확인 가능한 내용:**
- 기본 정보 (이름, 학년, 학교, 반, 전화번호)
- 입학테스트 결과 내역
- 단어시험 튜터링 세션 기록
- 과거 성적 데이터 연결 현황

## 7. 성적 통계 및 엑셀 내보내기

### 7-1. 성적 통계 확인

- **대시보드(`/`):** 전체 현황 요약 및 최근 채점 결과
- **`/analytics`:** 반 평균, 개인별 성적 추이, 석차 확인
- **`/word-submissions`:** 단어시험 전체 채점 결과 목록

### 7-2. 단어시험 결과 엑셀 내보내기

1. `/word-submissions` 페이지로 이동합니다.
2. 필요한 경우 시험명으로 필터링합니다.
3. **[엑셀 내보내기]** 버튼을 클릭합니다.
4. `word_test_results.xlsx` 파일이 다운로드됩니다.

**엑셀 파일에 포함되는 내용:**
- 제출일, 시험명, 학년, 학생이름
- 점수, 총 문항, 정답률(%)
- 문항별 학생 답안 및 정오 여부 (O/X/△)

> 정답률 85% 이상은 초록색, 65% 미만은 빨간색으로 표시됩니다.

### 7-3. 빨간펜 PDF 개별 다운로드

`/word-submissions` 목록에서 확정 완료된 채점 건을 클릭한 뒤, **[PDF 다운로드]** 버튼을 클릭합니다.

## 8. 자주 묻는 질문 (FAQ)

### Q1. NAS에 파일을 넣었는데 채점이 안 됩니다.

**확인 사항:**
1. 파일 형식이 올바른지 확인하세요 (JPG, PNG, WEBP, PDF).
2. 파일 이름에 학생 이름이 포함되어 있는지 확인하세요.
3. 파일 복사가 완전히 완료되었는지 확인하세요 (복사 중 감지 시 오류 발생 가능).
4. 해당 시험의 답지가 먼저 등록되어 있어야 합니다. 답지 폴더에 정답 파일이 있는지 확인하세요.
5. 그래도 안 된다면 NAS의 `미채점오류/` 폴더를 확인하세요. 오류 파일이 이동되어 있을 수 있습니다.

### Q2. AI가 학생 이름을 잘못 인식했습니다.

웹 `/word-submissions` 페이지에서 해당 채점 건을 열면 학생 이름을 직접 수정할 수 있습니다. 
수정 후 확정하면 됩니다.

학생 이름이 DB에 등록되어 있지 않은 경우 `미매칭` 상태로 처리됩니다. 
`/unmatched-submissions` 페이지에서 해당 건을 열고, **[학생 지정]** 기능으로 기존 학생과 연결할 수 있습니다.

### Q3. AI 채점 결과가 부정확합니다.

AI는 손글씨를 최대한 판독하지만, 필체가 매우 불명확한 경우 오류가 생길 수 있습니다. 
이런 경우 웹에서 직접 수정 후 확정하면 됩니다. 수정 내용은 엑셀 내보내기와 PDF에 모두 반영됩니다.

### Q4. 채점을 확정했는데 취소/수정하고 싶습니다.

`/word-submissions` 페이지에서 해당 채점 건을 열고 **[다시 열기]** 버튼을 누릅니다. 
상태가 `pending_review`로 돌아가며 다시 수정 후 확정할 수 있습니다.

### Q5. 빨간펜 PDF가 NAS에 저장되지 않았습니다.

NAS 연결이 끊어졌을 때 PDF 저장에 실패할 수 있습니다. 
이 경우 웹에서 **[PDF 다운로드]** 버튼으로 직접 다운받을 수 있습니다. 
NAS 연결 상태는 백엔드 로그에서 확인하거나 개발자에게 문의하세요.

### Q6. 엑셀로 학생을 일괄 등록했는데 일부가 누락됩니다.

엑셀 파일에 `이름`과 `학년` 컬럼이 반드시 있어야 합니다. 해당 행의 이름이나 학년이 비어 있으면 건너뛰게 됩니다.
업로드 후 응답 메시지에서 건너뛴 행 번호를 확인할 수 있습니다.

### Q7. 답지를 새 버전으로 교체하고 싶습니다.

NAS 답지 폴더에 새 PDF 파일을 넣으면 됩니다. 파일명이 다르면 새로운 시험으로 등록됩니다. 
기존 답지를 완전히 교체하고 싶다면 웹 `/word-tests` 페이지에서 기존 시험을 삭제한 뒤 새 파일을 등록하세요.

### Q8. 수학 OMR 채점에서 마킹이 인식되지 않습니다.

- 스캔 해상도를 200dpi 이상으로 높여 보세요.
- 마킹이 너무 연하거나 번진 경우 인식률이 낮을 수 있습니다.
- 웹 업로드 방식으로 다시 시도하거나 개발자에게 문의하세요.
